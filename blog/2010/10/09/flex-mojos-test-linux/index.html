Flex-mojos Tests on Linux

Oct 9th, 2010

I’ve run into issues testing on linux many times but Marvin from flex-mojos has lots of helpful info.

I had to do two things on my Ubuntu 10.04 box: * install flashplayer * install xvfb

Install Flashplayer
  • wget http://download.macromedia.com/pub/flashplayer/updaters/10/flash_player_10_linux_dev.tar.gz
  • tar -xvzf flash_player_10_linux_dev.tar.gz
  • cd flash_player_10_linux_dev/standalone/debugger
  • tar -xvzf flashplayer.tar.gz
  • put “flashplayer” on your PATH
Install xvfb
  • sudo apt-get install xvfb
  • flex-mojos runs xvfb-run, which is provided after installing xvfb

Hopefully I’ll remember this so I don’t have to look this page up in the future, but I’ll have it in case I do.
